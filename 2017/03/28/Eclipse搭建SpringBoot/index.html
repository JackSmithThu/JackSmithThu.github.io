1.在Eclipse中安装STS插件

进入Eclipse，选择【help】→【Eclipse Marketplace 】

搜索→【STS】,找到【SpringToolSuite】并下载。

2.新建一个项目

【file】->【new】->【project】

选择Spring Starter Project，然后【next】

给你的项目取个名字，我的叫【bootDemo】，点击【next】

添加你所需要的依赖，由于我们现在只是演示，就只添加【core】中的【AOP】和【web】中的【web】。选好之后，点击【next】->【Finish】。
