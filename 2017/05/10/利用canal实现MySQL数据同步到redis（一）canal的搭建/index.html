1.什么是cannal


cannal 的原理相对比较简单：



  1. canal 模拟 mysql slave 的交互协议，伪装自己为mysql slave，向 mysql master发送 dump 协议

  2. mysql master 收到 dump 请求，开始推送 binary log 给 slave (也就是 canal)

  3. canal 解析 binary log 对象(原始为 byte 流)
