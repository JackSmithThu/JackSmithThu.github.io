首先，什么是SCAtoolbox

SCAtoolbox是一个用于求解最优化问题的开源工具，使用matlab实现。
那么，什么是最优化问题呢？举个简单的例子：

z = x + y (0 < x < 1, 0 < y < 1)，求 z 的最大值

这就是一个简单版的最优化问题，其中：
z = x + y 称作目标函数
0 < x < 1, 0 < y < 1 称作约束条件

接下来，我们来看看SCAtoolbox怎么用

SCAtoolbox可以从以下链接获取。
http://www.alimirjalili.com/SCA.html
解压之后，用matlab打开SCAtoolbox.m即可食用。
