我们在相应请求时，主要分为两种情况：
  1. 我们想要返回一个字符串（可能是一个 json 串）。
  2. 我们想要返回一个页面。

对于前一种情况，我们可以使用 @ResponseBody 注解使返回的内容为我们想要返回的字符串。对于后一种情况，我们其实返回的是两样东西：

  1. 一个前端页面（我们称之为 templates）。
  2. 该页面所需要的数据（我们称之为 model）。
