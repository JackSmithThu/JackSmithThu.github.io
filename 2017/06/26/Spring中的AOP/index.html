1.什么是AOP

AOP（面向切面编程）给我们提供了很好的非入侵式（在不修改源代码的情况下）的手段为现有的类增加新的功能。本文通过演示 Spring AOP 是如何使用的来介绍什么是 AOP。

我们会首先演示如何使用基于注解的 Spring AOP，然后演示如何使用基于 XML 的 Spring AOP。最后对 Spring AOP 进行简要的总结。
