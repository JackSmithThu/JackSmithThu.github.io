1.打开文件

安装 vim，进入相应文件夹，输入

$ vim filename

即可打开相应的文件，我们这里打开一个名为 vimtest.txt 的文件。

K、J、H、K 分别对应着上下左右，当然，小键盘左边的方向键也是可以实现光标移动的。
